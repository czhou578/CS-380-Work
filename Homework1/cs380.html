CS 380 Web Programming

Introduces the basics of HTML, Javascript, and CSS, with further explorations of a Javascript framework. Also introduces security concepts with relation to web programming. Introduces how to setup a Node.js server.

  • I hope to learn how I can set up a personal node.js server.
  • I hope to understand the ways I can mitigate harmful viruses and attacks through the security concepts taught.
  • I've learned a lot of cool HTML tags that I haven't known about, including the code tag.
